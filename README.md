# 📦 JDBC CRUD Application

## 📌 Overview

This project demonstrates how to use **JDBC (Java Database Connectivity)** to connect a Java application with a relational database and perform basic database operations.

It is a simple yet structured implementation of **CRUD operations (Create, Read, Update, Delete)** using Java and Postgresql.

---

## 🚀 Features

* ✅ Establish database connection using JDBC
* ✅ Perform CRUD operations
* ✅ Use of `PreparedStatement` (prevents SQL injection)
* ✅ Clean and modular code structure
* ✅ Basic exception handling

---

## 🛠️ Tech Stack

* **Java**
* **JDBC**
* **Postgresql**
* **IntelliJ IDEA / Eclipse**

## ⚙️ Setup Instructions

### 1. Clone the repository

```
git clone <your-repo-link>
```

### 2. Open in IDE

Import the project into IntelliJ / Eclipse.

### 3. Configure Database

* Create a Postgresql database
* Update your credentials in the code:

```java
String url = "jdbc:mysql://localhost:3306/your_database";
String username = "your_username";
String password = "your_password";
```

### 4. Add Postgresql Connector

Download and add Postgresql JDBC Driver (Connector/J) to your project.
